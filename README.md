# HaruLing Content Automation

K-콘텐츠 뉴스를 자동으로 수집하고, Claude AI가 분석하여 주간 보고서를 생성하는 자동화 도구입니다.

## 🎯 기능

- 📰 RSS 뉴스 수집 (Google News + 한국 언론사)
- 🤖 Claude AI 분석 (주간 보고서 생성)
- 📝 Obsidian 자동 저장

## 🚀 설정

1. .env.example → .env 복사
2. GOOGLE_API_KEY 입력
3. Claude Desktop에서 자동 실행

## 📁 구조

- config/ - 설정 파일
- CLAUDE.md - 자동화 지시사항
- README.md - 프로젝트 설명
