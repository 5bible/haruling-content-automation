# SKILL: 뉴스 분석 및 보고서 작성 (analyze-content)

## 📌 스킬 설명
collect-news에서 수집된 뉴스를 분석하여 HaruLing 주간 트렌드 보고서를 작성합니다.
CLAUDE.md의 Step 2에서 read_file로 이 파일을 읽은 후 실행됩니다.

## 🔧 입력값
- collect-news 스킬이 컨텍스트에 보관한 JSON 기사 목록 (별도 파일 읽기 불필요)
- 현재 날짜 (주차 번호 계산용)

## 📋 실행 절차

### 1단계: 주차 번호 계산
현재 날짜를 기준으로 ISO 8601 연간 주차 번호를 계산합니다.
- 기준: 해당 연도의 첫 번째 목요일이 포함된 주가 Week 1
- 예시: 2026-04-21 → Week 17, 연도 2026
- 계산 결과를 {N}과 {year}에 바로 적용

### 2단계: 보고서 작성
아래 분석 지시에 따라 수집된 기사를 분석하고 보고서를 작성합니다.

**분석 지시 (당신은 K-콘텐츠 마케팅 전문가입니다)**
HaruLing 한국어 학습 앱의 사용자(K-콘텐츠에 관심 있는 외국인 학습자)를 위해
수집된 뉴스를 분석하고 아래 구조로 보고서를 한국어로 작성하세요:

1. 이번 주 트렌드 핵심 요약 (3~5줄)
2. 카테고리별 분석
   - 드라마/영화
   - 가요/뮤직
   - 아이돌/엔터테인먼트
3. 외국인 학습자 언어·문화 학습 포인트 (2~3가지)
4. HaruLing 콘텐츠 아이디어 (3~5가지, 구체적 소재 + 활용 방법 포함)
5. 참고 기사 목록 (표 형식: 제목 | 출처 | 링크)

## 📤 출력값 형식
{N}과 {year}에 실제 계산된 값을 채워 아래 형식으로 완성된 Markdown을 작성합니다.
완성된 보고서 전문을 Claude 컨텍스트에 보관합니다.

```markdown
---
title: "Week 17 - K-Content Trends [2026]"
date: 2026-04-21
tags: [HaruLing, K-콘텐츠, 주간트렌드]
---
```

## ⚠️ 주의사항
- 작성 언어: 한국어 (기사 원제는 원문 병기 가능)

- 보고서 내 {N}과 날짜는 플레이스홀더가 아닌 실제 값으로 작성
