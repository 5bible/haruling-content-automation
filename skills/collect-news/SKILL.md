# SKILL: RSS 뉴스 수집 (collect-news)

## 📌 스킬 설명
외국인 한국어 학습자가 흥미롭고 재미있게 읽을 수 있는
K-콘텐츠 뉴스를 RSS 피드에서 수집합니다.
CLAUDE.md의 Step 1에서 read_file로 이 파일을 읽은 후 실행됩니다.

## 🎯 수집 목표
"K-콘텐츠에 관심 있는 외국인이 읽으면 흥미롭고 재미있을 뉴스"를 수집합니다.
HaruLing 기업 분석이나 비즈니스 뉴스가 아닙니다.

## 🔧 수집 설정
config/news_sources.yaml의 collection_settings를 따릅니다:
- items_per_week : 10 (최대 수집 기사 수)
- max_age_days   : 7  (수집 기준 기간)

## ✅ 수집 대상 (이런 기사를 우선 선택)
- 인기 드라마·영화 줄거리, 명장면, 결말 반응
- K-pop 신곡·앨범·MV 발매 및 반응
- 아이돌 활동, 해외 콘서트, 팬미팅
- 한국 버라이어티쇼·예능에서 화제가 된 장면
- 한국 배우·가수의 해외 활동 및 글로벌 반응
- 외국인들 사이에서 유행하는 한국 문화 트렌드
- 넷플릭스 등 글로벌 플랫폼의 K-콘텐츠 화제작

## ❌ 수집 제외 (이런 기사는 건너뜀)
- 엔터테인먼트 기업의 실적·투자·인수합병 뉴스
- 연예인 법적 분쟁·소송·계약 관련 뉴스
- 기획사 경영·조직 변경 뉴스
- HaruLing 앱이나 한국어 교육 업계 비즈니스 뉴스
- 정치·경제 뉴스 (K-콘텐츠와 직접 관련 없는 것)

## 📋 실행 절차

1. config/news_sources.yaml 파일을 read_file로 읽어 RSS URL 전체 목록 확인
2. 각 URL에 순서대로 접근하여 피드 파싱
3. 각 기사에서 아래 정보 추출
   - title    : 기사 제목
   - source   : 언론사명
   - link     : 원문 URL
   - published: 발행일시
   - summary  : 요약 (없으면 본문 첫 2문장 사용)
4. 발행일 기준 최근 7일 이내 기사만 필터링
5. 수집 대상/제외 기준으로 필터링
6. 중복 제거 (동일 제목 또는 동일 링크 기준)
7. 최대 10개 선별 — 균형 기준:
   - 드라마/영화, 가요/K-pop, 아이돌/엔터테인먼트 카테고리 균형 유지
   - 우선순위: 외국인 반응이 뜨거운 기사 > 최신 기사 > 요약이 풍부한 기사

## 📤 출력값
아래 형식의 JSON 목록을 Claude 컨텍스트에 보관:

```json
[
  {
    "title": "기사 제목",
    "source": "출처명",
    "link": "https://...",
    "published": "2026-04-21",
    "summary": "기사 요약 내용"
  }
]
```

## ⚠️ 예외 처리
- 특정 피드 접근 실패 시: 해당 피드 건너뛰고 나머지 계속 수집
- 수집 기사 5개 미만 시: 사용자에게 경고 메시지 출력 후 분석 단계 진행
- 요약 없는 기사: 본문 첫 2문장으로 대체
- 인코딩: UTF-8
